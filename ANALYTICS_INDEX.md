### Events (21 total)
- **User:** Signup, Login, Onboarding (3)
- **Video:** Started, Completed, Skipped (3)
- **Learning:** Word Clicked/Saved/Reviewed/Mastered (4)
- **Game:** Started, Completed, Score (3)
- **Payment:** Checkout, Payment, Cancellation (3)
- **Engagement:** Daily Active, Session Start/End (3)
- **Content:** Article Read, Podcast Listened (2)

### User Properties (11 total)
- Language Level, Days Since Signup
- Total Videos, Total Words
- Current Streak, Longest Streak
- Subscription Status, Device Type
- Target Language, Native Language, Last Active

### Metrics (20+ total)
- **Engagement:** DAU, MAU, DAU/MAU, Sessions
- **Retention:** D1, D7, D14, D30
- **Conversion:** Free‚ÜíPaid, Checkout‚ÜíPayment
- **Business:** MRR, Churn, LTV, CAC
- **Technical:** API time, Errors, DB queries

### Errors (4 types)
- JavaScript, API, Database, Payment

## üöÄ Quick Actions

### Common Tasks

**Track a video view:**
```javascript
const { track } = require('./lib/analytics-integration');
track.videoStarted(userId, videoId, { title: 'Spanish 101' });
```

**Capture an error:**
```javascript
const { captureError } = require('./lib/analytics-integration');
captureError.api(error, req, { videoId });
```

**Get user analytics:**
```bash
curl http://localhost:3001/api/analytics/dashboard/USER_ID
```

**Check system metrics:**
```bash
curl http://localhost:3001/api/analytics/system/metrics
```

**View in Mixpanel:**
1. Go to https://mixpanel.com/
2. Select project
3. View Events/Insights/Funnels

**View in Sentry:**
1. Go to https://sentry.io/
2. Select project
3. View Issues/Performance
